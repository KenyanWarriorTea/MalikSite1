# MalikSite — Judge0 учебная платформа (MVP)

Сайт для двух ролей без регистрации:
- **Учитель**: добавляет задания и смотрит ответы учеников
- **Ученик**: открывает задания и отправляет решения

Проверка решений выполняется через **Judge0 API**.

## Запуск в Docker

1. Клонировать репозиторий
2. Выполнить:
   ```bash
   docker-compose up --build
   ```
3. Открыть сайт: http://localhost:8000

## Как пользоваться

1. На главной странице ввести имя и выбрать роль (учитель/ученик)
2. Учитель добавляет задание (название, описание, expected output, language_id)
3. Ученик отправляет код
4. Учитель в своей панели видит:
   - имя ученика
   - статус проверки Judge0
   - код ученика
   - stdout/stderr
